# 🚀 TaskFlow Pro

A production-ready task management platform showcasing modern full-stack development with enterprise-grade DevOps practices.

## 🛠️ Tech Stack

- **Frontend:** Angular 18, React 18
- **Backend:** Spring Boot 3.3, Java 21
- **Database:** MySQL 8, Redis 7
- **DevOps:** Docker, Kubernetes, GitHub Actions

## 🏃 Quick Start

Start infrastructure with Docker Compose, then run backend with Maven.
